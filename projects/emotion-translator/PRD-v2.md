# 家长情绪翻译器 - PRD v2

## 1. 项目背景与目标

**一句话描述**：一个帮助家长将负面情绪吐槽转化为积极管教话术的移动端 Web 应用。

**目标用户**：0-12 岁孩子的家长（25-40 岁父母）

**核心价值**：解决家长情绪爆发时不知如何好好说话的问题，通过"打字降温"机制改善亲子沟通。

**UI 风格**：温暖安抚风，蓝绿色(#4ECDC4)主调，米黄色(#FFF8E7)背景，圆润卡片设计。

## 6. TTS 接入方案（火山引擎）

### 6.1 为什么选择火山引擎？
- 豆包同款技术，超自然语音合成
- 专为中文优化，情感表达丰富
- 免费额度充足（每月 100 万字符）
- 国内稳定访问，无需代理

### 6.2 接入步骤

**Step 1：注册火山引擎账号**
- 访问 https://www.volcengine.com/
- 完成实名认证

**Step 2：开通语音合成服务**
- 进入「语音技术」→「语音合成大模型」
- 点击「立即开通」

**Step 3：创建应用获取凭证**
- 进入「应用管理」→「创建应用」
- 获取：
  - `AppID`
  - `AccessToken`
  - `Cluster`（一般为 `volcano_tts`）

**Step 4：查看音色列表**
- 推荐亲子场景的音色：

| 音色 ID | 名称 | 适用场景 |
|---------|------|---------|
| BV007 | 亲切女声 | 温柔妈妈（默认推荐） |
| BV056 | 阳光男声 | 智慧爸爸 |
| BV005 | 活泼女声 | 亲切姐姐 |
| BV051 | 奶气萌娃 | 低龄孩子场景 |

### 6.3 API 接入代码示例

```typescript

  const appid = '你的 AppID';
```

## 7. AI 提示词设计（v2 优化）

```
你是一位专业的亲子沟通教练，拥有儿童心理学和正面管教的专业背景。

将家长的负面情绪吐槽转化为积极、建设性的管教话术。

输出格式（严格遵循）：
---对孩子说的话---
[2-3句话，温暖坚定，可直接朗读给孩子听]

---给家长的提示---
[解释为什么这么说的教育心理学原理，帮助家长理解]

转换原则：
1. 先认可家长的情绪
2. 不给孩子贴标签，聚焦具体行为
3. 提供具体可执行的建议
4. 语气温暖而坚定
5. 话术控制在2-3句话
```

## 9. 示例

**输入**：
> "你怎么又把衣服扔得到处都是！"

**输出**：
```
---对孩子说的话---
"我看到地上有衣服，这让我有点担心。我们一起把衣服放回衣柜，好吗？"

---给家长的提示---
先描述事实（地上有衣服）而非指责（你怎么又），用"我们"而非"你"，把命令变成邀请，孩子更容易配合。
```
