# 家长情绪翻译器 - PRD

## 1. 项目背景与目标

**一句话描述**：一个帮助家长将负面情绪吐槽转化为积极管教话术的移动端 Web 应用。

**目标用户**：0-12 岁孩子的家长（25-40 岁父母）

**核心价值**：解决家长情绪爆发时不知如何好好说话的问题，通过"打字降温"机制改善亲子沟通。

**UI 风格**：温暖安抚风，蓝绿色(#4ECDC4)主调，米黄色(#FFF8E7)背景，圆润卡片设计。

---

## 2. 技术栈约束

- 框架：React 18 + TypeScript
- 样式：Tailwind CSS
- AI：Moonshot (Kimi) API
- 语音：Web Speech API
- 存储：LocalStorage (MVP)

---

## 3. 核心功能模块 (MVP)

**必须有**：
1. 孩子档案管理（添加/查看/选择）
2. 情绪翻译（输入吐槽→AI转译）
3. 语音朗读
4. 历史记录（最近20条）

**暂不做**：登录同步、统计分析、分享、小程序

---

## 4. 页面结构与用户流程

**用户流程**：打开App → 选择孩子 → 输入吐槽 → AI翻译 → 听语音/复制 → 查看历史

**页面清单**：

| 路由 | 页面 | 核心组成 |
|-----|------|---------|
| / | 首页 | Logo、核心理念、开始按钮 |
| /kids | 孩子管理 | 孩子列表、添加表单、选择按钮 |
| /translate | 翻译页 | 输入框、翻译按钮、结果卡片、语音按钮、复制按钮 |
| /history | 历史记录 | 倒序列表、每条显示孩子名+原文+译文+时间 |

**交互细节**：
- 输入框自动增高，最大300字
- 翻译按钮加载时显示"正在理解您的情绪..."
- 语音播放显示波形动画

---

## 5. 数据结构

**Child**：id, name, age, gender, createdAt

**Translation**：id, childId, childName, originalText, translatedText, createdAt

---

## 6. AI 提示词

你是一位专业的亲子沟通教练。将家长的负面情绪吐槽转化为积极、建设性的管教话术。

原则：
1. 先认可家长情绪
2. 不给孩子贴标签
3. 提供可执行建议
4. 语气温暖而坚定
5. 控制在2-3句话

只输出转化后的话术，不要解释。

---

## 7. 示例

输入："你怎么又把衣服扔得到处都是！"
输出："我看到地上有衣服，这让我有点担心。我们一起把衣服放回衣柜，好吗？"
